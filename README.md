# 課堂教學整合網 (Smart Classroom Portal)

專為中小學課堂教學設計的高效、美觀且具備高度自訂性的靜態網頁整合門戶。適合投影於教室大螢幕智慧觸控白板、平板或個人電腦使用。

## 🌟 核心特色

1. **整合式教學資源與課堂工具**
   - **常用教學資源**：內建三大教科書平台快速連結（**康軒數位高手**、**翰林行動大師**、**南一數位資源入口網**）。
   - **課堂工具**：整合實用互動工具（**班級工具**與**課堂表現登記系統**）。
   - 所有連結均於新分頁開啟，不會覆蓋教學網頁主介面。

2. **第 1 至 6 冊教學資料分頁（高度自訂）**
   - 依學期分為第一冊至第六冊，分頁切換流暢。
   - 預設提供「備課教材」、「簡報與影音」、「學習單與試卷」之精美佔位空間。
   - **自訂書籤功能**：老師可以直接點擊「新增連結」卡片，儲存自訂的 Google Drive、網頁連結或教案路徑。資料透過瀏覽器 `localStorage` 永久儲存，方便下次上課直接點擊使用。

3. **課堂專用數位時鐘**
   - 頁首醒目的即時時鐘，顯示時、分、秒，並提供完整的年月日與星期幾資訊，方便掌控課堂進度。

4. **隨堂備忘錄**
   - 提供教師隨手記錄課堂聯絡簿、待辦事項或下堂課進度。
   - 具備自動儲存機制（Auto-save to LocalStorage），打字即存，重啟網頁或重新開機資料皆不會遺失。

5. **極致美學視覺**
   - 使用 Outfit 與 Noto Sans TC 精緻字型。
   - 採用磨砂玻璃視覺風格（Glassmorphic UI）及流暢的懸停發光微粒子效果，大螢幕呈現質感極佳。

## 🚀 如何使用與部署

### 本地直接執行
直接於瀏覽器中開啟 [index.html](index.html) 即可開始使用。

### 部署至 GitHub Pages（推薦）
本專案已設定 GitHub Pages 自動部署。建立 GitHub 儲存庫並 Push 後，即可透過 GitHub Pages 網址線上開啟，在任何裝置上登入瀏覽器皆可同步使用您的自訂配置。

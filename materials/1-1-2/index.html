重點 1：整數的加法運算

重點整理與敘述

  • 同號數相加：
    • 兩個同正數相加，其和為正數，值等於兩數絕對值相加。例如：\(5 + 2 = 7\)。
    • 兩個同負數相加，其和為負數，值等於兩數絕對值相加後冠上負號。例如：\((-5) + (-2) = -(5 + 2) = -7\)。
  • 異號數相加：
    • 兩個異號數相加時，結果的性質符號取自絕對值較大的那數，其值為絕對值較大者減去較小者。例如：\(5 + (-2) = 3\)；\((-5) + 2 = -3\)。
  • 特別性質：
    • 與 0 相加：任何數加 0 其值不變，即 \(a + 0 = a\)。
    • 相反數相加：兩互為相反數的整數相加，其和恆為 0。即 \(a + (-a) = 0\)。
  • 加法運算規律：
    • 交換律：\(a + b = b + a\)。
    • 結合律：\((a + b) + c = a + (b + c)\)。

形成性評量 (觀念挑戰)

Q1. 計算式子 \((-9) + (-7)\) 的值為何？

解析：這是兩個負數（同號數）相加。性質符號與原兩數相同為「－」，數值為兩數絕對值相加：\((-9) + (-7) = -(9 + 7) = -16\)。

Q2. 計算式子 \((-12) + 7\) 的值為何？

解析：這是異號數相加。因為負數的絕對值較大（\(|-12| > |7|\)），所以結果的符號為「－」，其值等於大絕對值減小絕對值：\((-12) + 7 = -(12 - 7) = -5\)。

重點 2：整數的減法運算

重點整理與敘述

  • 減法核心定義：
    • 減去一個數，等於加上這個數的相反數。即對任意整數 \(a, b\)，有：
      \[a - b = a + (－b)\]
  • 物理情境（溫度變化量）：
    • 溫度變化計算公式：\(\text{最後溫度} - \text{原來溫度} = \text{溫度變化}\)。
    • 最後溫度高於原來溫度（大減小）：溫度上升，變化量為正數。例如：\(10 - 6 = 4\)（升高 4 度）。
    • 最後溫度低於原來溫度（小減大）：溫度下降，變化量為負數。例如：\(2 - 5 = -3\)（降低 3 度）。
  • 減去負整數：
    • 減去一個負數，等於加上該數的絕對值（正整數）。例如：\(8 - (-3) = 8 + 3 = 11\)。

形成性評量 (減法挑戰)

Q1. 計算式子 \(125 - (-25)\) 的值為何？

解析：減去一個負數等於「加上該數的相反數」，因此：\(125 - (-25) = 125 + 25 = 150\)。

Q2. 計算式子 \((-63) - 37\) 的值為何？

解析：減去一個數等於加上它的相反數，所以：\((-63) - 37 = (-63) + (-37) = -(63 + 37) = -100\)。

重點 3：去括號規則與加減混合運算

重點整理與敘述

  • 去括號基本規則：
    • 括號前面是「＋」號：去括號後，原來括號內各項的性質符號保持不變！
      \[a + (b - c) = a + b - c\]
    • 括號前面是「－」號：去括號後，原來括號內各項性質符號必須變號（＋變－、－變＋）！
      \[a - (b - c) = a - b + c\]
  • 去括號的重要公式：
    • \(-(a + b) = -a - b\)
    • \(-(a - b) = -a + b\)
  • 絕對值運算順序：
    • 如果算式中含有絕對值符號，應優先處理絕對值內部的運算，再求其絕對值，最後才做其它的運算。

形成性評量 (混合運算挑戰)

Q1. 利用去括號規則，計算式子 \(299 - (396 + 299)\) 的值為何？

解析：因為括號前面是「－」號，去括號展開後括號內的各項性質符號都要變號：
\(299 - (396 + 299) = 299 - 396 - 299\)。
利用加法交換律與結合律，先將同數相減：
\(299 - 299 - 396 = 0 - 396 = -396\)。因此答案為 C。

Q2. 若 \(a, b\) 為兩個整數，關於式子 \(-(a - b)\) 去括號的結果，下列何者完全正確？

解析：括號前面是負號，展開後各項都必須變號：\(-(a - b) = -a + b\)。根據加法交換律，\(-a + b\) 調整順序後即等於 \(b - a\)。因此答案為 C。

重點 4：數線上兩點的距離與中點

重點整理與敘述

  • 數線兩點間距離 (線段長度)：
    • 若數線上有 \(A(a)\) 與 \(B(b)\) 兩點，則其距離（記為 \(AB\)）等於兩點座標差的絕對值：
      \[AB = |a - b| = |b - a|\]
    • 白話說法：數線兩點的距離，就是「大數減去小數」的值。
  • 中點座標 (Midpoint)：
    • 如果 \(C(c)\) 在 A、B 之間，且到 A、B 的距離相等（\(AC = BC\)），我們稱 C 為 \(AB\) 的中點。
    • 中點的座標為 A、B 兩數相加除以 2（即算術平均數）：
      \[c = \frac{a + b}{2}\]

形成性評量 (幾何距離挑戰)

Q1. 數線上有 \(C(-14)\)、\(D(-9)\) 兩點，則其距離 \(\overline{CD}\) 為何？

解析：兩點的距離等於兩座標相減的絕對值：\(\overline{CD} = |-14 - (-9)| = |-14 + 9| = |-5| = 5\)。
另外也可以直接用「大數減小數」：\(-9 - (-14) = -9 + 14 = 5\)。距離恆為正數，答案為 D。

Q2. 數線上有 \(A(-3)\)、\(B(7)\) 兩點，若 \(C(c)\) 為 \(AB\) 的中點，則 \(c\) 是多少？

解析：中點座標計算方式為兩點座標相加除以 2：
\(c = \frac{-3 + 7}{2} = \frac{4}{2} = 2\)。
也可以先算距離 \(AB = |7 - (-3)| = 10\)，一半距離為 \(10 / 2 = 5\)，故中點在 \(A(-3)\) 右邊 5 單位：\(-3 + 5 = 2\)。答案為 A。

1-2 整數的加減 課程核心回顧

加減法快速檢索卡

  • 同號加法：同符號，數值相加。
    \( (+a) + (+b) = +(a+b) \)
    \( (-a) + (-b) = -(a+b) \)
  • 異號加法：取大絕對值符號，大減小。
    \( (-5) + 2 = -(5-2) = -3 \)
  • 減法變加法：減一個數等於「加相反數」。
    \( a - b = a + (-b) \)
    \( a - (-b) = a + b \)
  • 去括號規則：
    括號前是「＋」不變號：\( +(a-b) = a-b \)
    括號前是「－」要變號：\( -(a-b) = -a+b \)

生活中的加減故事

海拔與溫度計差

臺灣玉山頂上在冬天出現低溫 \(-10^\circ\text{C}\)，而平地臺北出現了 \(15^\circ\text{C}\)。高溫比低溫高了幾度呢？我們可以用大數減去小數：

\( 15 - (-10) = 15 + 10 = 25 \)

哇！差了 \(25\) 度呢！「減去一個負數」就像是拉開兩個溫度計的垂直差距，等同於直接相加，學會整數加減後，日常生活中的計量、海拔、甚至是理財記帳都變得更容易掌控了！

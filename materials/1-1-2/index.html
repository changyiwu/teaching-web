1-2 整數的加減

重點 1：整數的加法運算

重點整理與敘述

  • 同號數相加：
    • 兩個同正數相加，其和為正數，值等於兩數絕對值相加。例如：\(5 + 2 = 7\)。
    • 兩個同負數相加，其和為負數，值等於兩數絕對值相加後冠上負號。例如：\((-5) + (-2) = -(5 + 2) = -7\)。
  • 異號數相加：
    • 兩個異號數相加時，結果的性質符號取自絕對值較大的那數，其值為絕對值較大者減去較小者。例如：\(5 + (-2) = 3\)；\((-5) + 2 = -3\)。
  • 與 0 或相反數相加：
    • 與 0 相加：任何數加 0 其值不變，即 \(a + 0 = a\)。
    • 相反數相加：兩互為相反數的整數相加，其和恆為 0。即 \(a + (-a) = 0\)。

形成性評量 (觀念挑戰)

Q1. 計算式子 \((-8) + (-6)\) 的值為何？

解析：這是兩個負數（同號數）相加。性質符號與原兩數相同為「－」，數值為兩數絕對值相加：\((-8) + (-6) = -(8 + 6) = -14\)。

Q2. 計算式子 \((-15) + 9\) 的值為何？

解析：這是異號數相加。因為負數的絕對值較大（\(|-15| > |9|\)），所以結果的符號為「－」，其值等於大絕對值減小絕對值：\((-15) + 9 = -(15 - 9) = -6\)。

重點 2：加法運算規律

重點整理與敘述

  • 加法交換律：
    • 若 \(a, b\) 為整數，則任意兩數相加時，調換順序其和不變：
      \[a + b = b + a\]
    • 例如：\((-2) + 5 = 3\)，而 \(5 + (-2) = 3\)。
  • 加法結合律：
    • 若 \(a, b, c\) 為整數，三個數相加時，先算前兩數或後兩數，其結果皆相同：
      \[(a + b) + c = a + (b + c)\]
    • 例如：\([5 + (-3)] + 1 = 2 + 1 = 3\)，而 \(5 + [(-3) + 1] = 5 + (-2) = 3\)。
  • 簡化計算技巧：
    • 在加減混合運算中，可利用結合律先將相反數或相加為整十、整百、整千的同號數結合計算，能大幅簡化運算過程。

運算規律精妙應用示範

觀察以下算式如何透過交換位置與重新結合，在不使用計算機的情況下快速得出解答：

範例：計算 \((-34) + 2508 + (-66)\)

  1. 第一步（交換律）：將相近的數移在一起
    \(\rightarrow 2508 + (-34) + (-66)\)
  2. 第二步（結合律）：把兩個同負號的數先結合
    \(\rightarrow 2508 + [(-34) + (-66)]\)
  3. 第三步（速算）：\((-34) + (-66) = -100\)
    \(\rightarrow 2508 + (-100)\)
  4. 第四步（答案）：得到最終答案為 \(2408\)！

形成性評量 (規律挑戰)

Q1. 計算式子 \((-34) + 2508 + (-66)\) 的值為何？

解析：運用加法的交換律與結合律，先將同為負數的兩項結合：\((-34) + (-66) = -100\)。再與 \(2508\) 相加：\(2508 + (-100) = 2408\)。因此答案為 A。

Q2. 計算式子 \(245 + (-87) + (-245)\) 的值為何？

解析：利用交換律將互為相反數的 \(245\) 與 \(-245\) 擺放在一起先做相加：\(245 + (-245) = 0\)。最後加上 \(-87\)：\(0 + (-87) = -87\)。因此答案為 D。

重點 3：整數的減法運算

重點整理與敘述

  • 減法核心定義：
    • 減去一個數，等於加上這個數的相反數。即對任意整數 \(a, b\)，有：
      \[a - b = a + (-b)\]
  • 物理情境（溫度變化量）：
    • 溫度變化計算公式：\(\text{最後溫度} - \text{原來溫度} = \text{溫度變化}\)。
    • 最後溫度高於原來溫度（大減小）：溫度上升，變化量為正數。例如：\(10 - 6 = 4\)（升高 4 度）。
    • 最後溫度低於原來溫度（小減大）：溫度下降，變化量為負數。例如：\(2 - 5 = -3\)（降低 3 度）。
  • 減去負整數：
    • 減去一個負數，等於加上該數的絕對值（正整數）。例如：\(8 - (-3) = 8 + 3 = 11\)。

形成性評量 (減法挑戰)

Q1. 計算式子 \(142 - (-58)\) 的值為何？

解析：減去一個負數等於「加上該數的相反數（即正數）」，因此：\(142 - (-58) = 142 + 58 = 200\)。因此答案為 A。

Q2. 計算式子 \((-45) - 35\) 的值為何？

解析：減去一個數等於加上它的相反數，所以：\((-45) - 35 = (-45) + (-35) = -80\)。因此答案為 D。

重點 4：去括號規則與加減混合運算

重點整理與敘述

  • 去括號基本規則：
    • 括號前面是「＋」號：去括號後，原來括號內各項的性質符號保持不變！
      \[a + (b - c) = a + b - c\]
    • 括號前面是「－」號：去括號後，原來括號內各項性質符號必須變號（＋變－、－變＋）！
      \[a - (b - c) = a - b + c\]
  • 去括號重要公式：
    • \(-(a + b) = -a - b\)
    • \(-(a - b) = -a + b\)
  • 簡化混合運算：
    • 善用去括號規則，可以讓我們自由決定是先運算括號內部的數，還是先展開消去相同項目，達到口算效果。

形成性評量 (混合運算挑戰)

Q1. 利用去括號規則，計算式子 \(388 - (475 + 388)\) 的值為何？

解析：因為括號前面是「－」號，去括號展開後括號內的各項性質符號都要變號：
\(388 - (475 + 388) = 388 - 475 - 388\)。
利用加法交換律與結合律，先將同數相減：
\(388 - 388 - 475 = 0 - 475 = -475\)。因此答案為 C。

Q2. 若 \(x, y\) 為兩個整數，關於式子 \(-(x - y)\) 去括號的結果，下列何者完全正確？

解析：括號前面是負號，展開後各項都必須變號：\(-(x - y) = -x + y\)。根據加法交換律，\(-x + y\) 調整順序後即等於 \(y - x\)。因此答案為 C。

重點 5：含絕對值的算式運算

重點整理與敘述

  • 絕對值優先原則：
    • 如果算式中含有絕對值符號，應優先處理絕對值內部的運算，將結果取絕對值（化為正數或 0）之後，再進行其他的四則運算。
  • 絕對值與括號的本質差異：
    • 括號前有負號可以根據去括號規則變號：如 \(-(5 - 8) = -5 + 8 = 3\)。
    • 絕對值前有負號，**絕對不能**直接去括號變號！必須先算內部：如 \(-|5 - 8| = -|-3| = -3\)。這兩者運算邏輯完全不同！
  • 經典範例對比：
    • \(2 - |-5| = 2 - 5 = -3\) (先取絕對值再減)
    • \(2 - (-5) = 2 + 5 = 7\) (去括號變號)

絕對值防錯避坑指南

絕對值代表的是「距離」，其值恆為大於或等於零的數。請注意以下重點防錯機制：

常犯錯誤：
錯誤算式：\(|(-3) + (-4)| = |-3| + |-4| = 3 + 4 = 7\) （❌，絕對值不能這樣拆開！）
正確解法：
應先將內部運算完畢再取值：
\(|(-3) + (-4)| = |-7| = 7\) （\(|a + b|\) 與 \(|a| + |b|\) 僅在同號時值相同，若一正一負則會出錯，因此務必先算內部！）

形成性評量 (絕對值挑戰)

Q1. 計算式子 \(|-35| - |-85| - 12\) 的值為何？

解析：含有絕對值先處理絕對值：\(|-35| = 35\)，\(|-85| = 85\)。
原式可化簡為 \(35 - 85 - 12 = -50 - 12 = -62\)。因此答案為 B。

Q2. 計算式子 \(40 + |(-56) + 16| - 15\) 的值為何？

解析：必須優先算出絕對值內部的算式：\((-56) + 16 = -40\)。
再求其絕對值：\(|-40| = 40\)。
原式變為：\(40 + 40 - 15 = 65\)。因此答案為 C。

重點 6：數線上兩點的距離與中點

重點整理與敘述

  • 數線兩點間距離 (線段長度)：
    • 若數線上有 \(A(a)\) 與 \(B(b)\) 兩點，則其距離（記為 \(AB\)）等於兩點座標差的絕對值：
      \[AB = |a - b| = |b - a|\]
    • 白話說法：數線兩點的距離，就是「大數減去小數」的值。
  • 中點座標 (Midpoint)：
    • 如果 \(C(c)\) 在 A、B 之間，且到 A、B 的距離相等（\(AC = BC\)），我們稱 C 為 \(AB\) 的中點。
    • 中點的座標為 A、B 兩數相加除以 2（即算術平均數）：
      \[c = \frac{a + b}{2}\]

形成性評量 (幾何距離挑戰)

Q1. 數線上有 \(C(-18)\)、\(D(-7)\) 兩點，則其距離 \(\overline{CD}\) 為何？

解析：兩點的距離等於兩座標相減的絕對值：\(\overline{CD} = |-18 - (-7)| = |-18 + 7| = |-11| = 11\)。
另外也可以直接用「大數減小數」：\(-7 - (-18) = -7 + 18 = 11\)。距離恆為正數，答案為 D。

Q2. 數線上有 \(A(-5)\)、\(B(9)\) 兩點，若 \(C(c)\) 為 \(\overline{AB}\) 的中點，則 \(c\) 是多少？

解析：中點座標計算方式為兩點座標相加除以 2：
\(c = \frac{-5 + 9}{2} = \frac{4}{2} = 2\)。因此答案為 A。

1-2 整數的加減 課程核心回顧

加減法快速檢索卡

  • 同號加法：同符號，數值相加。
    \( (+a) + (+b) = +(a+b) \)
    \( (-a) + (-b) = -(a+b) \)
  • 異號加法：取大絕對值符號，大減小。
    \( (-5) + 2 = -(5-2) = -3 \)
  • 減法變加法：減一個數等於「加相反數」。
    \( a - b = a + (-b) \)
    \( a - (-b) = a + b \)
  • 去括號規則：
    括號前是「＋」不變號：\( +(a-b) = a-b \)
    括號前是「－」要變號：\( -(a-b) = -a+b \)

生活中的加減故事

海拔與溫度計差

臺灣玉山頂上在冬天出現低溫 \(-10^\circ\text{C}\)，而平地臺北出現了 \(15^\circ\text{C}\)。高溫比低溫高了幾度呢？我們可以用大數減去小數：

\( 15 - (-10) = 15 + 10 = 25 \)

哇！差了 \(25\) 度呢！「減去一個負數」就像是拉開兩個溫度計的垂直差距，等同於直接相加，學會整數加減後，日常生活中的計量、海拔、甚至是理財記帳都變得更容易掌控了！
